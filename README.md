# 🌤️ SkinCast

A smart skincare recommendation platform that provides personalized skincare suggestions based on real-time weather conditions. The application helps users maintain healthy skin by recommending products, routines, and home remedies according to environmental factors such as temperature, humidity, and weather conditions.

## 🚀 Features

- 🌍 Real-time weather integration using APIs
- 💄 Personalized skincare product recommendations
- 🧴 Customized skincare routines
- 🏠 Home remedy suggestions
- 🤖 AI-powered chatbot assistance
- 📱 Responsive and user-friendly interface
- 🔍 Dynamic weather-based recommendations

## 💡 How It Works

1. User enters location or uses current location.
2. Application fetches real-time weather data.
3. Weather conditions are analyzed.
4. Personalized skincare recommendations are generated.
5. Users receive product suggestions, routines, and home remedies.
